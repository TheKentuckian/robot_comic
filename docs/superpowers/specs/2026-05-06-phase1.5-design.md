# Phase 1.5 ‚Äî Repeat Visitor Identity Matching

**Goal:** On conversation start, the robot detects face presence, asks the person's name, and fuzzy-matches it against stored crowd-work sessions. If matched, it loads the returning visitor's profile and opens with a warm callback instead of a cold start.

**Approach:** New `Greet` tool in `profiles/don_rickles/`. Two-action design: `scan` (face detection + head sweep) and `identify` (fuzzy name match against stored sessions). Tools remain decoupled ‚Äî `Greet` returns matched profile data and the LLM calls `crowd_work(action="update")` to load it.

**Spec:** This document.

## Tool Interface

### `Greet` tool

**`action="scan"`**

Detects face presence in the current camera frame using MediaPipe. If no face is found, executes a square sweep before giving up.

Returns:
```json
{"face_detected": true}
```
or
```json
{"no_subject": true}
```
or (MediaPipe unavailable ‚Äî fail-open):
```json
{"face_detected": true, "note": "MediaPipe unavailable, assuming face present"}
```

**`action="identify", name="<spoken name>"`**

Scans `.rickles_sessions/` for JSON files modified within the last 30 days. Extracts `name` from each, fuzzy-matches against the spoken name using `difflib.SequenceMatcher` with threshold `0.75`. Takes the highest-scoring match above threshold.

Returns on match:
```json
{
  "returning": true,
  "name": "Tony",
  "last_seen": "2026-05-04",  // from session JSON's last_updated field; fallback to file mtime if null
  "profile": {
    "job": "engineer",
    "hometown": "Pittsburgh",
    "details": ["nervous laugh", "wrinkled shirt"]
  },
  "callbacks": ["name + job: Tony, engineer", "job + visual: engineer + nervous laugh"],
  "load_instruction": "Call crowd_work action=update with this profile before proceeding."
}
```

Returns on no match:
```json
{"returning": false}
```

**Parameters schema:**

```python
parameters_schema = {
    "type": "object",
    "properties": {
        "action": {
            "type": "string",
            "enum": ["scan", "identify"],
            "description": (
                "scan: detect face presence; executes head sweep if no face found initially. "
                "identify: fuzzy-match a spoken name against stored sessions from the last 30 days."
            ),
        },
        "name": {
            "type": "string",
            "description": "The name spoken by the person. Required for action='identify'.",
        },
    },
    "required": ["action"],
}
```

## Face Detection & Head Sweep

### Face detection

Uses `mediapipe.solutions.face_detection.FaceDetection(min_detection_confidence=0.5)`.

Frame conversion: BGR‚ÜíRGB via numpy slice `frame[..., ::-1]` ‚Äî no OpenCV dependency.

```python
def _detect_face(frame: NDArray) -> bool:
    rgb = frame[..., ::-1].copy()
    with mp_face_detection.FaceDetection(min_detection_confidence=0.5) as detector:
        results = detector.process(rgb)
        return bool(results.detections)
```

### Square sweep (triggered when initial scan finds no face)

Sweep positions in order: `"left"` ‚Üí `"up"` ‚Üí `"right"` ‚Üí `"front"` (always resets to center at end).

At each position:
1. `MoveHead(direction=<position>)`
2. `await asyncio.sleep(deps.motion_duration_s + 0.2)`
3. Capture frame from `camera_worker`
4. Run MediaPipe face detection
5. Face found ‚Üí stop sweep, return `{"face_detected": True}`

If sweep completes with no face found ‚Üí return `{"no_subject": True}`.

`MoveHead` direction support is audited during implementation; unsupported directions are skipped gracefully.

### MediaPipe availability

`mediapipe` is an optional project dependency (`mediapipe_vision` extras). If not installed:
- Fail-open: return `{"face_detected": True, "note": "MediaPipe unavailable, assuming face present"}`
- Rationale: better to proceed to a conversation than to silently abort every session on a machine without MediaPipe installed.

## Fuzzy Name Matching

```python
import difflib

MATCH_THRESHOLD = 0.75
SESSION_WINDOW_DAYS = 30

def _fuzzy_match(query: str, candidates: list[tuple[str, Path]]) -> tuple[str | None, Path | None, float]:
    """Return (best_name, best_path, score) for the highest match above threshold."""
    best_name, best_path, best_score = None, None, 0.0
    q = query.lower().strip()
    for name, path in candidates:
        score = difflib.SequenceMatcher(None, q, name.lower().strip()).ratio()
        if score > best_score:
            best_score, best_name, best_path = score, name, path
    if best_score >= MATCH_THRESHOLD:
        return best_name, best_path, best_score
    return None, None, best_score
```

Session scanning:
1. Glob `.rickles_sessions/session_*.json`
2. Filter to files with `mtime` within 30 days
3. Load each JSON, extract `name` field ‚Äî skip if `None` or unreadable
4. Run fuzzy match across all candidates
5. If match: load full session state from the matched path; extract `profile` fields; build callbacks using the same algorithm as `CrowdWork._build_callbacks` (duplicated in `greet.py` ‚Äî do not import `CrowdWork`)

## Updated `instructions.txt` Opening Sequence

Replaces the existing `## OPENING SEQUENCE` section:

```
## OPENING SEQUENCE

The moment a conversation starts:

1. Speak an opening line:
   - "Hold on ‚Äî don't move. Let me get a good look at you..."
   - "Okay. Who do we have here."
   - "Oh, wonderful. Look what just walked in."

2. Call greet with action="scan".
   - If no_subject: true ‚Äî the room is empty. Improvise:
     "I know you're out there. I can hear you breathing."
     "Come on, show yourself. I've been practising."
   - If face_detected: true ‚Äî ask their name in character:
     "And who exactly are we dealing with here? Give me a name."

3. When they answer, call greet with action="identify" and the spoken name.
   - If returning: true ‚Äî open WARM. Use the callbacks. Skip roast entirely for the open.
     Load their history first: call crowd_work with action="update" and the returned profile fields.
     Then open with a callback:
     "Well look who it is ‚Äî [name]. Still doing that [job]? Beautiful. Just beautiful."
     "Oh no. Not [name] again. I thought we had an agreement."
   - If returning: false ‚Äî open COLD. Call roast. Proceed with normal crowd-work flow.
```

## Error Handling

| Condition | Behaviour |
|-----------|-----------|
| `camera_worker is None` | `{"error": "Camera not available"}` |
| `camera_worker.get_latest_frame()` returns `None` | `{"error": "No frame available"}` |
| MediaPipe not installed | Fail-open: `{"face_detected": True, "note": "MediaPipe unavailable, assuming face present"}` |
| No sessions in 30-day window | `{"returning": False}` |
| All candidates have `name: null` | `{"returning": False}` |
| Corrupt/unreadable session file | Skip, log warning, continue |
| `name` missing on `identify` | `{"error": "name is required for action='identify'"}` |
| `MoveHead` returns error during sweep | Log warning, continue to next sweep position |
